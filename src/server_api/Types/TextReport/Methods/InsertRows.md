# TextReport.InsertRows(TextReport, long, long) մեթոդ

```c#
public Task InsertRows(TextReport textReport, long begin = 0, long rowCount = 0)
```

Ավելացնում է textReport հաշվետվության նշված տողերը ընթացիկ հաշվետվությունում։ 

**Պարամետրեր**

| Անվանում        | Տվյալների տիպ        | Լռությամբ արժեք | Նկարագրություն |
|-----------------|----------------------|----------------|----------------|
| textReport      | TextReport           | -              | Այն հաշվետվությունը, որի տողերը պետք է ավելացվեն ընթացիկ հաշվետվությունում։ |
| begin           | long                 | 0              | `textReport` հաշվետվության տողի համար, որից սկսած ավելացվում են տողերը ընթացիկ հաշվետվությունում։ Արժեք չփոխանցելու դեպքում ավելացումը կսկսվի 0 համարի տողից։ |
| rowCount        | long                 | 0              | Ավելացվող տողերի քանակը՝ սկսած `begin` համարի տողից։ Արժեք չփոխանցելու դեպքում ավելացվելու են բոլոր տողերը՝ սկսած `begin` համարի տողից։ |
