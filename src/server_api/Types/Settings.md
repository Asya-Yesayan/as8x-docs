# Settings դաս

## Ներածություն

Նախատեսված է 8x սերվիսի կարգավորումների հետ աշխատանքը ապահովելու համար։

## Հատկություններ

| Անվանում | Նկարագրություն |
|----------|----------------|
| [CurrentServiceId](Settings/Properties/CurrentServiceId.md) | Վերադարձնում է ընթացիկ սերվիսի id-ն։ |
