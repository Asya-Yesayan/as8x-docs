# ProgressPhase դաս

## Ներածություն

Այս դասը նախատեսված է կատարման պրոգրեսի փուլի նկարագրման համար։

## Հատկություններ


| Անվանում | Տվյալների տիպ | Լռությամբ արժեք | **Նկարագրություն** |
| --- | --- | --- | --- |
| Id | int | - | Կատարման պրոգրեսի փուլի համարը: |
| Name | string | - |  Կատարման պրոգրեսի փուլի անվանումը: |
| StartDate | string | - |  Կատարման պրոգրեսի առաջադրանքի մեկնարկի ամսաթիվը: |
| EndDate | string | - |  Կատարման պրոգրեսի առաջադրանքի ավարտի ամսաթիվը: |
| Duration | double? | - |  Կատարման պրոգրեսի փուլի տևողությունը: |
| Row | int | -1 |  Կատարման պրոգրեսի փուլում ընթացիկ պահին մշակված տողերի քանակը: |
| Total | int | -1 |  Կատարման պրոգրեսի փուլի ընթացքում մշակման ենթակա տողերի քանակը: |
| DisableCancel | bool | - |  Կատարման պրոգրեսի փուլը կարելի է ընդհատել, թե ոչ: |
| Overwritable | bool | true |  Կատարման պրոգրեսի փուլը կարելի է անվանափոխել, թե ոչ: |
