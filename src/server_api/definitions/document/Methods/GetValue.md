# Document.GetValue(string) մեթոդ

## Նկարագիր

**Դաս՝** [Document](../Document.md)

```c#
public T GetValue<T>(string name, T defaultValue)
```

Վերադարձնում է փաստաթղթի նշված ներքին անունով դաշտի (ռեկվիզիտի) արժեքը՝ բերված T տիպի։

**Պարամետրեր**

| Անվանում | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն |
| --- | --- | --- | --- |
| name | string | - | Փաստաթղթի դաշտի ներքին անունը։ |
| defaultValue | T | - | Եթե փաստաթղթում նշված ներքին անունով դաշտը բացակայում է, ապա վերադարձնում է այս պարամետրում տրված արժեքը։ |
