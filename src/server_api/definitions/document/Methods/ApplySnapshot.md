## Նկարագիր

**Համարժեքը 4x-ում՝** AsDoc.ApplySnapshot

public void ApplySnapshot(DocumentSnapshot snapshot)

`snapshot` պարամետրում եղած փաստաթղթի քեշավորված պատկերի արժեքները վերագրում է փաստաթղթի դաշտերին և աղյուսակներին։
