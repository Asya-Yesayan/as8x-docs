# UserReportService սերվիս

## Ներածություն

UserReportService դասը նախատեսված է օգտագործողի հաշվետվությունների (`UserReport`) հետ աշխատանքը (կատարում, հեռացում, ․․․) ապահովելու համար։

## Մեթոդներ

| Անվանում | Նկարագրություն |
|----------|----------------|
| [ExecuteOld(string, CancellationToken)](Methods/ExecuteOld.md) | Կատարում է 4x-ական օգտագործողի հաշվետվությունը և վերադարձնում է արդյունքը որպես `IList`։ |
