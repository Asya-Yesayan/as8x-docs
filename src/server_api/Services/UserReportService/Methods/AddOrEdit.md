# UserReportService.AddOrEdit(ReportModel) մեթոդ

```c#
public Task<int> AddOrEdit(ReportModel reportModel)
```

ReportModel.Id ներքին նույնականացման համարով (id) օգտագործողի հաշվետվության առկայության դեպքում թարմացնում է հաշվետվության նկարագրությունը, հակառակ դեպքում ստեղծում է նորը։ Վերադարձնում է օգտագործողի հաշվետվության ներքին նույնականացման համարը (id)։ 

**Պարամետրեր**

| Անվանում     | Տվյալների տիպ   | Լռությամբ արժեք | Նկարագրություն |
|--------------|-----------------|----------------|----------------|
| reportModel  | ReportModel     | -              | Օգտագործողի հաշվետվության նկարագրությունը։ |
