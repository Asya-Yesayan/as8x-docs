# JobProcessService.Cancel(int) մեթոդ

```c#
public Task<bool> Cancel(int executionId)
```

Փորձում է ընդհատել առաջադրանքի կատարումը և վերադարձնում է bool արժեք, որը ցույց է տալիս՝ ընդհատումը հաջողվել է, թե ոչ։ Կատարման պրոգրեսի բացակայության կամ առաջադրանքի ավարտված լինելու դեպքում ընդհատումը չի կատարվի։

**Պարամետրեր**

| Անվանում        | Տվյալների տիպ        | Լռությամբ արժեք | Նկարագրություն |
|-----------------|----------------------|----------------|----------------|
| executionId     | int                  | -              | Առաջադրանքի կատարման id-ն։ |
