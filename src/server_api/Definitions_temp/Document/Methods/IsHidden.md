# Document.IsHidden(string) մեթոդ

## Նկարագիր

**Դաս՝** [Document](../Document.md)

```c#
public bool IsHidden(string name)
```

Ստուգում է փաստաթղթի նշված ներքին անունով դաշտի (ռեկվիզիտի) անտեսանելի լինելու հայտանիշը (H ատրիբուտի առկայությունը)։ Եթե [փաստաթղթի նկարագրությունում](https://armsoft.github.io/as4x-docs/HTM/ProgrGuide/Defs/doc.html) նշված ներքին անունով դաշտը բացակայում է, ապա առաջանում է սխալ։

**Պարամետրեր**

| Անվանում | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն |
| --- | --- | --- | --- |
| name | string | - | Փաստաթղթի դաշտի ներքին անունը։ |
