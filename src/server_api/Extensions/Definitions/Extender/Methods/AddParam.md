## Նկարագիր

**Դաս՝** [Extender](../ds_extender.md)

```c#
public void AddParam(string name, 
                     string description, 
                     FieldType fieldType)
```

Ընդլայնման միջոցով ավելացող նոր պարամետրի հատկությունները սահմանելու համար անհրաժեշտ է կոնստրուկտորում կանչել այս մեթոդը։

**Պարամետրեր**

| Անվանում     | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն        |
|--------------|---------------|----------------|------------------------|
| name         | string        | —              | Պարամետրի ներքին անունը։     |
| description  | string        | —              | Պարամետրի հայերեն նկարագրությունը։ Անհրաժեշտ է փոխանցել ANSI կոդավորմամբ։ |
| fieldType    | FieldType     | —              | Պարամետրի համակարգային տիպը։ |
