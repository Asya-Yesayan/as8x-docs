public Task DeleteScheduledJob(int scheduledId)

**Պարամետրեր**

| Անվանում | Տեսակ | **Նկարագրություն** |
| --- | --- | --- |
| scheduledId | int | Հերթագրված առաջադրանքի id-ն։ |
