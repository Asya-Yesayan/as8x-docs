# WebServicesManager սերվիս

## Մեթոդներ

| Անվանում | Նկարագրություն |
|----------|----------------|
| [Add(WebServiceInfo)](Methods/Add.md) | Ավելացնում է նոր վեբ ծառայություն։ |
| [Delete(string)](Methods/Delete.md) | Հեռացնում է վեբ ծառայությունը: |

| [Get(string, string)](Methods/Get.md) | Վերադարձնում է վեբ ծառայությունների նկարագրությունների ցանկը՝ ըստ տրված `id`-ի կամ `role`-ի։ Եթե պարամետրերը նշված չեն, վերադարձնում է բոլոր հասանելի վեբ ծառայությունները։ |
