## Ներածություն

AppLogService դասը նախատեսված է հատուկ իրադարձությունների (տարբերակի փոփոխություն, Debug-ի գործարկում) մանրամասների տվյալների պահոցում գրանցվելու ֆունկցիոնալությունը ապահովելու համար։

## Մեթոդներ

|----------|----------------|
| [Write(AppLogInfo)](AppLogService/Methods/Write.md)) | Գրանցում է հատուկ իրադարձության (տարբերակի փոփոխություն, Debug-ի գործարկում...) մանրամասները տվյալների պահոցում։ |
| [Write(string, string, string, int, int)](AppLogService/Methods/Write.md) | Գրանցում է հատուկ իրադարձության (տարբերակի փոփոխություն, Debug-ի գործարկում...) մանրամասները տվյալների պահոցում։ |
