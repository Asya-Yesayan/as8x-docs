## Ներածություն

AppLogService դասը նախատեսված է հատուկ իրադարձությունների (տարբերակի փոփոխություն, Debug-ի գործարկում) մանրամասների տվյալների պահոցում գրանցվելու ֆունկցիոնալությունը ապահովելու համար։
