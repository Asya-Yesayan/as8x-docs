## Նկարագիր

**Դաս՝** [IAttachmentService](../../IAttachmentService.md)

```c#
public Task Copy(int copyFromISN, 
                 int copyToISN, 
                 bool updateExisted = true)
```

Պատճենում է տրված փաստաթղթի կցված ֆայլերը մեկ այլ փաստաթղթի մեջ։

**Պարամետրեր**

| Անվանում        | Տվյալների տիպ        | Լռությամբ արժեք | Նկարագրություն |
|-----------------|----------------------|----------------|----------------|
| copyFromISN     | int                  | -              | Փաստաթղթի ներքին նույնականացման համար, որից պետք է պատճենվեն կցված ֆայլերը: |
| copyToISN       | int                  | -              | Փաստաթղթի ներքին նույնականացման համար, որտեղ պետք է պատճենվեն կցված ֆայլերը: |
| updateExisted   | bool                 | true           | `copyToISN` ներքին նույնականացման համարով փաստաթղթին կցված ֆայլերի առկայության դեպքում ֆայլերի վերագրանցման հայտանիշ։ |
