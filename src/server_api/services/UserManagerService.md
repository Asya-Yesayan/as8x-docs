## Ներածություն

UserManagerService դասը նախատեսված է օգտագործողների ղեկավարումը (ավելացում, խմբագրում, հեռացում․․․) ապահովելու համար։

## Մեթոդներ

| Անվանում | Նկարագրություն |
|----------|----------------|
| [GetAccessibleRoles(short)](UserManagerService/Methods/GetAccessibleRoles.md) | Վերադարձնում է օգտագործողին նշանակված դերերի ներքին անունների ցուցակը։ |
| [GetUnhiddenUsers()](UserManagerService/Methods/GetUnhiddenUsers.md) | Վերադարձնում է համակարգի բոլոր անտեսանելի [օգտագործողների նկարագրությունների](../types/UserDescription.md) ցուցակը։ |
