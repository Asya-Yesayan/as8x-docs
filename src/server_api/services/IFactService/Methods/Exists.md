## Նկարագիր

**Համարժեքը 4x-ում՝** [AsKernel.ExistsFact](https://armsoft.github.io/as4x-docs/HTM/ProgrGuide/Functions/Functions/AccManagement/ExistsFact.html)

**Դաս՝** [IFactService](../../IFactService.md)

```c#
public Task<bool> Exists(int baseIsn = 0, 
                         int objectIsn = 0, 
                         string accounting = "", 
                         string operation = "", 
                         bool useArchive = true);
```

Ստուգում է նշված հաշվառող կամ հիմք փաստաթղթով գործառույթների առկայությանը։

Մեթոդի կանչի ժամանակ անհրաժեշտ է արժեք փոխանցել `baseIsn` կամ `objectIsn` պարամետրերից գոնե մեկին։

**Պարամետրեր**

| Անվանում        | Տվյալների տիպ        | Լռությամբ արժեք | Նկարագրություն |
|-----------------|----------------------|----------------|----------------|
| baseIsn         | int                  | 0              | Հիմքային փաստաթղթի ներքին նույնականացման համար։ |
| objectIsn       | int                  | 0              | Հաշվառող փաստաթղթի ներքին նույնականացման համար։ |
| accounting      | string               | ""             | Հաշվառման կոդը։ |
| operation       | string               | ""             | Գործողությունների կոդերի ցանկ։ <br> Եթե ցանկը սկսվում է “+” նշանով, ապա հաշվի են առնվում գործողությունների բոլոր կոդերը, որոնք թվարկվում են նրանից հետո։ <br>  “-“ նշանի դեպքում անտեսվում են այն գործողությունները, որոնց կոդերը թվարկված են ցանկի մեջ։ Գործողությունների կոդերը թվարկվում են մեկը մյուսի հետևից բացատների միջոցով։ <br> Ցանկը նաև կարող է պարունակել առանց նշանի միակ տարր:  |
| useArchive      | bool                 | true           | Արխիվացված գործառնության որոնման հայտանիշ։ |
