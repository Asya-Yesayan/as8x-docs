# JobProcessService.GetProgress(int) մեթոդ

```c#
public Progress GetProgress(int executionId)
```

Վերադարձնում է առաջադրանքի կատարման պրոգրեսը՝ ըստ առաջադրանքի կատարման id-ի։ Կատարման պրոգրեսի բացակայության կամ առաջադրանքի ավարտված լինելու դեպքում վերադարձնում է null:

**Պարամետրեր**

| Անվանում        | Տվյալների տիպ        | Լռությամբ արժեք | Նկարագրություն |
|-----------------|----------------------|----------------|----------------|
| executionId     | int                  | -              | Առաջադրանքի կատարման id-ն։ |
