# JobProcessService.HasProgress(int) մեթոդ

```c#
public bool HasProgress(int executionId)
```

Վերադարձնում է bool տիպի արժեք, որը ցույց է տալիս առաջադրանքի կատարման պրոգրեսի առկայությունը։ 

**Պարամետրեր**

| Անվանում        | Տվյալների տիպ        | Լռությամբ արժեք | Նկարագրություն |
|-----------------|----------------------|----------------|----------------|
| executionId     | int                  | -              | Առաջադրանքի կատարման id-ն։ |
