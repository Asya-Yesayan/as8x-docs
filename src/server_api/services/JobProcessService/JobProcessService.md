# JobProcessService սերվիս

## Ներածություն

JobProcessService դասը նախատեսված է առաջադրանքների կատարման պրոգրեսի հետ աշխատանքը ապահովելու համար։

## Մեթոդներ

| Անվանում | Նկարագրություն |
|----------|----------------|
| [GetProgress](Methods/GetProgress.md) | Վերադարձնում է առաջադրանքի կատարման պրոգրեսը՝ ըստ առաջադրանքի կատարման id-ի։ Կատարման պրոգրեսի բացակայության կամ առաջադրանքի ավարտված լինելու դեպքում վերադարձնում է null: |
| [Cancel](Methods/Cancel.md) | Փորձում է ընդհատել առաջադրանքի կատարումը և վերադարձնում է bool արժեք, որը ցույց է տալիս՝ ընդհատումը հաջողվել է, թե ոչ։ Կատարման պրոգրեսի բացակայության կամ առաջադրանքի ավարտված լինելու դեպքում ընդհատումը չի կատարվի։ |
| [HasProgress](Methods/HasProgress.md) | Վերադարձնում է bool տիպի արժեք, որը ցույց է տալիս առաջադրանքի կատարման պրոգրեսի առկայությունը։ |
