## Ներածություն

IHolidaysService դասը նախատեսված է աշխատանքային և տոն/ոչ աշխատանքային օրերի որոշման համար։

## Մեթոդներ

| Անվանում | Նկարագրություն |
|----------|----------------|
| [GetNextWorkDay(DateTime)](IHolidaysService/Methods/GetNextWorkDay.md) | Վերադարձնում է տրված ամսաթվին հաջորդող առաջին աշխատանքային օրը: |
| [GetPreviousWorkDay(DateTime)](IHolidaysService/Methods/GetPreviousWorkDay.md) | Վերադարձնում է տրված ամսաթվին նախորդող առաջին աշխատանքային օրը: |
| [IsHoliday(DateTime)](IHolidaysService/Methods/IsHoliday.md) | Ստուգում է արդյոք տրված ամսաթիվը տոն/ոչ աշխատանքային օր է։ |
| [IsWorkday(DateTime)](IHolidaysService/Methods/IsWorkday.md) | Ստուգում է արդյոք տրված ամսաթիվը աշխատանքային օր է։ |
