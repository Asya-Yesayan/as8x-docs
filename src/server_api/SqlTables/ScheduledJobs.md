# job.SCHEDULEDJOBS աղյուսակ

Այս աղյուսակը նախատեսված է հերթագրված առաջադրանքների (job) ինֆորմացիայի պահպանման համար։ 

Ժամանակացույցով գործարկվող առաջադրանքները համակարգը հերթագրում են ավտոմատ կերպով, իսկ Ձեռքով հերթագրելու համար անհրաժեշտ է կանչել «Ադմինիստրատորի ԱՇՏ» → «Առաջադրանքներ» → «Առաջադրանքների ձևանմուշներ» հաշվետվության **Հերթագրել կատարման** կոնտեքստային ֆունկցիան։
