# RolesModel

Այս դասը նախատեսված է համակարգի դերի նկարագրման համար։

```c#
public class RolesModel
{
    public string Name { get; set; }

    public string ArmenianCaption { get; set; }

    public string EnglishCaption { get; set; }
}
```

## Հատկություններ

| Անվանում        | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն           |
| --------------- | ------------- | --------------- | ------------------------ |
| Name            | string        |      -           | Դերի ներքին անունը։      |
| ArmenianCaption | string        |      -           | Դերի հայերեն անվանումը։  |
| EnglishCaption  | string        |      -           | Դերի անգլերեն անվանումը։ |
