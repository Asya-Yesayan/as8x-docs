## Ներածություն

Այս դասը նախատեսված է 8x սերվիսի կարգավորումների, պարամետրերի հետ աշխատանքը ապահովելու համար։

## Հատկություններ

| Անվանում | Նկարագրություն |
|----------|----------------|
| [CurrentServiceId](Settings/CurrentServiceId.md) | Վերադարձնում է ընթացիկ սերվիսի id-ն։ |
