```c#
public void AddFooter(string footer)
```

Ավելացնում է նոր տող հաշվետվության ստորին հատվածում։

**Պարամետրեր**
| Անվանում        | Տվյալների տիպ        | Լռությամբ արժեք | Նկարագրություն |
|-----------------|----------------------|----------------|----------------|
| footer          | string               | -              | Ավելացվող տողի տեքստ։ |
