# NotificationGroup դաս

## Ներածություն

Այս դասը նախատեսված է ծանուցման խմբի նկարագրման համար։

## Հատկություններ


| Անվանում    | Տվյալների տիպ                              | Լռությամբ արժեք | **Նկարագրություն**   |
| ----------- | ------------------------------------------ | --------------- | -------------------- |
| Id          | int                                        | —               | Ծանուցման խմբի կոդը (id):   |
| Name        | string                                     | —               | Ծանուցման խմբի անվանումը։    |
| Description | string                                     | —               | Ծանուցման խմբի նկարագրությունը։  |
| Subscribers | Dictionary<NotificationType, List<string>> | []              | Dictionary տիպի օբյեկտ, որտեղ բանալին [ծանուցման եղանակն](NotificationType.md) է (էլ․ փոստով, ․․․), իսկ արժեքը՝ նշված եղանակով ծանուցվող օգտագործողների իդենտիֆիկատորների ցուցակը (էլ․ փոստի հասցե, username, ...):   |
