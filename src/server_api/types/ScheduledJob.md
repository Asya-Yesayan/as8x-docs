# ScheduledJob դաս

## Ներածություն

Այս մոդելը պարունակում է հերթագրված առաջադրանքի նկարագրությունը։

## Հատկություններ

| Անվանում         | Տեսակ                 | Նկարագրություն |
|------------------|----------------------|----------------|
| Id               | int                  | Հերթագրված առաջադրանքի id-ն։ |
| ISN              | int                  | Առաջադրանքի ձևանմուշը նկարագրող փաստաթղթի ներքին նույնականացման համարը (isn): |
| JobCode          | string               | Առաջադրանքի կոդը։ |
| StartDate        | DateTime             | Առաջադրանքի նախատեսված մեկնարկի ամսաթիվը/ժամանակը։ |
| TargetService    | string               | Այն վեբ ծառայության (սերվիսի) id-ն, որի վրա գործարկվելու է տվյալ առաջադրանքը։ <br> Վեբ ծառայությունները սահմանվում են «Ադմինիստրատորի աշտ» -> «Համակարգային աշխատանքներ» -> **«8X Web ծառայություններ»** հաշվետվության **Ավելացնել** կոնտեքստային ֆունկցիայով։ Սահմանվող վեբ ծառայության դերը անհրաժեշտ է նշել **JobExecuter**՝ ծառայության միջոցով առաջադրանքների գործարկումը թույլատրելու նպատակով: |
| Suid             | short                | Առաջադրանքը կատարող օգտագործողի id-ն։ |
| Activation       | byte                 | Առաջադրանքի գործարկման տեսակը։ <br> **0** - Ձեռքով, <br> **1** - ժամանակացույց, <br> **2** - Առաջադրանքից հետո, <br> **3** - Իրադարձությունից հետո։ |
| State            | byte                 | Առաջադրանքի ընթացիկ վիճակը։ <br> **0** - Հերթագրված,  <br> **1** - Կատարման ընթացքում,  <br> **2** - Բարեհաջող կատարված,  <br> **3** - Ընդհատված կատարում։ |
| Comment          | string               | Առաջադրանքի կատարման վիճակի մասին մեկնաբանություն։ |
| JobName          | string               | Առաջադրանքի անվանումը։ |
| Steps            | List<JobStep>        | Առաջադրանքի կատարման քայլերի [նկարագրությունների](JobStep.md) ցուցակը։ |
| NotifyOn         | [JobNotification](JobNotification.md)  | Առաջադրանքի ծանուցման [տարբերակը](JobNotification.md)։ Ծանուցումներ կստանան այն ծանուցման խմբերի օգտագործողները, որոնց կոդերը նշված են 8XJOBNOTIFICATION համակարգային պարամետրում։  |
