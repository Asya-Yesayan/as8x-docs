Այս enum-ը նախատեսված է Html տպելու ձևանմուշների լրացման ժամանակ պատկերների տեղադրման եղանակի սահմանման համար։

```c#
public enum HtmlImageOption
{
    None = -1,
    RelativePath,
    AbsolutePath,
    Base64
}
```

**Արժեքների բազմություն**

| Անվանում     | Արժեք | Նկարագրություն                                                                          |
| ------------ | ----- | --------------------------------------------------------------------------------------- |
| None         | -1    | Պատկերները չեն տեղադրվելու։                                                             |
| RelativePath | 0     | Տրված ճանապարհից ստացվում և տեղադրվում է միայն պատկերի անվանումը (օր.՝ "HtmlImage.jpg") |
| AbsolutePath | 1     | Տեղադրվում է տրված ճանապարհը ամբողջությամբ (օր.՝ "C:\HtmlImage.jpg")                    |
| Base64       | 2     | Տեղադրվում է PrintTemplateSubstitutionImage-ում տրված File base64 ֆորմատով։             |
