## Ներածություն

Այս դասը նախատեսված է առաջադրանքի կատարման քայլի ծանուցման կարգավորումների սահմանման համար։

## Հատկություններ

| Անվանում    | Տվյալների տիպ                              | Լռությամբ արժեք | **Նկարագրություն**   |
| ----------- | ------------------------------------------ | --------------- | -------------------- |
| NotifyOn    | JobNotification                            | —               | Առաջադրանքի կատարման քայլի ծանուցման [տարբերակը](JobNotification.md)։   |
| Groups      | List<string>                               | —               | Այն ծանուցման խմբերի կոդերի ցուցակը, որոնք ծանուցվելու են քայլի կատարման ընթացքում։     |
| AttachFiles | string                                     | —               |  Ծանուցումում կցված ֆայլերի առկայության հայտանիշ։ Այս պահին ֆունկցիոնալությունը իրականացված չէ։ |
