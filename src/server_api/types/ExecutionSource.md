---
title: "ExecutionSource Enum" 
---

Այս enum-ը նախատեսված է կատարման պրոգրեսի ստեղծման աղբյուրի սահմանման համար։

## Դաշտեր

| Անվանում | Նկարագրություն |
|----------|----------------|
| Unknown | Անհայտ։ |
| DataSource | Տվյալների աղբյուրի կատարում: |
| Document | Փաստաթղթի գրանցում/հեռացում: |
| DataProcessingRequest | Տվյալների մշակման հարցում (DPR) կատարում: |
| Job | Առաջադրանքի կատարում։ |
