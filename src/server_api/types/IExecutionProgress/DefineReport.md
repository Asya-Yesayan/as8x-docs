---
title: IExecutionProgress.DefineReport(string, string, bool) մեթոդ
---

```c#
public void DefineReport(string armenianCaption, 
                        string englishCaption = "", 
                        bool useFormatting = false);
```

Ստեղծում է տեքստային հաշվետվություն՝ 120 լայնությամբ հատվածով (ֆրագմենտով)։ Մեթոդի կրկնակի կանչի դեպքում առաջանում է սխալ։

Առաջադրանքի կատարման ընթացքում մեթոդի կանչի դեպքում տեքստային հաշվետվությունը մնում է անփոփոխ։

**Պարամետրեր**

| Անվանում        | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն                                                        |
| --------------- | ------------- | --------------- | --------------------------------------------------------------------- |
| armenianCaption | string        | -               | Տեքստային հաշվետվության հայերեն անվանումը։                                  |
| englishCaption  | string        | ""              | Տեքստային հաշվետվության անգլերեն անվանումը։                     |
| useFormatting   | bool          | false           | Տեքստային հաշվետվությունում տեքստի ձևաչափեր օգտագործելու հայտանիշը։ |
