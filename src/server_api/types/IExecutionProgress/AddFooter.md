```c#
public void AddFooter(string footer);
```

Ավելացնում է նոր տող տեքստային հաշվետվության ստորին հատվածում։ Առաջադրանքի կատարման ժամանակ մեթոդի կանչը անտեսվում է։

**Պարամետրեր**

| Անվանում | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն                                       |
| -------- | ------------- | --------------- | ---------------------------------------------------- |
| footer   | string        | -               | Ավելացվող տողի տեքստը։ |
