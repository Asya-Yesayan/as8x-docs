title: IExecutionProgress.AddErrorEvent(string, Exception exception =) մեթոդ
---

```c#
public void AddErrorEvent(string message, Exception exception = null);
```

Ավելացնում է նոր տող տեքստային հաշվետվությունում։ Եթե մեթոդը կանչվել է առաջադրանքի կատարման ժամանակ, ապա Seq-ում ավելացվում է նոր իրադարձություն՝ նշված հաղորդագրությամբ, սխալի հաղորդագրությամբ և Error լոգավորման մակարդակով։

**Պարամետրեր**

| Անվանում  | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն                                        |
| --------- | ------------- | --------------- | ----------------------------------------------------- |
| message   | string        | -               | Ավելացվող տեքստը։                              |
| exception | Exception     | null            | Exception տիպի սխալ։ |
