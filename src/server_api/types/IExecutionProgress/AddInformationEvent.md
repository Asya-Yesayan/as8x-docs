```c#
public void AddInformationEvent(IEnumerable<string> messages);
```

Ավելացնում է նոր տողեր տեքստային հաշվետվությունում, եթե մեթոդը չի կանչվել առաջադրանքի կատարման ժամանակ։ Առաջադրանքի կատարման ընթացքում կանչի դեպքում միայն Seq-ում ավելացվում է նոր իրադարձություն՝ նշված հաղորդագրություններով և Information լոգավորման մակարդակով, իսկ տեքստային հաշվետվությունը մնում է անփոփոխ։

| -------- | ------------- | --------------- | ---------------------------------------------------- |
| messages  | IEnumerable<string>        | -               | Ավելացվող տեքստերը։ |
