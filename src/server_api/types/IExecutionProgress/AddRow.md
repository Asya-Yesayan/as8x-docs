```c#
public void AddRow(string message);
```

Ավելացնում է նոր տող տեքստային հաշվետվությունում։ Եթե մեթոդը կանչվել է առաջադրանքի կատարման ժամանակ, ապա Seq-ում ավելացվում է նոր իրադարձություն՝ նշված հաղորդագրությամբ և Information լոգավորման մակարդակով։

**Պարամետրեր**

| Անվանում | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն                                       |
| -------- | ------------- | --------------- | ---------------------------------------------------- |
| message  | string        | -               | Ավելացվող տեքստը։ |
