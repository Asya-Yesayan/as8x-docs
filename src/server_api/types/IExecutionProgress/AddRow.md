title: IExecutionProgress.AddRow(string) մեթոդ

```c#
public void AddRow(string message);
```

Ավելացնում է նոր տող տեքստային հաշվետվությունում, եթե մեթոդը չի կանչվել առաջադրանքի կատարման ժամանակ։ Առաջադրանքի ընթացքում կանչի դեպքում միայն Seq-ում ավելացվում է նոր իրադարձություն՝ նշված հաղորդագրությամբ և Information լոգավորման մակարդակով, իսկ տեքստային հաշվետվությունը մնում է անփոփոխ։

**Պարամետրեր**

| Անվանում | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն                                       |
| -------- | ------------- | --------------- | ---------------------------------------------------- |
| message  | string        | -               | Ավելացվող տեքստը։ |
