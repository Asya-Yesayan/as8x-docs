# IGridRow.this հատկություն

```c#
public object this[string colName] { get; set; }
```

Վերադարձնում կամ նշանակում է աղյուսակի ընթացիկ տողի նշված ներքին անունով սյան արժեքը։

**Պարամետրեր**

| Անվանում        | Տվյալների տիպ                          | Լռությամբ արժեք | Նկարագրություն |
|-----------------|----------------------------------------|----------------|---------------|
| colName         | string                                 | —              | Սյան ներքին անունը (կոդը)։ |
