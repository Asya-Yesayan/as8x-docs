### IGridRow.this հատկություն

```c#
public object this[int index] { get; set; }
```

Վերադարձնում կամ նշանակում է աղյուսակի ընթացիկ տողի նշված ինդեքսով սյան արժեքը։

**Պարամետրեր**

| Անվանում        | Տվյալների տիպ                          | Լռությամբ արժեք | Նկարագրություն |
|-----------------|----------------------------------------|----------------|---------------|
| index           | int                                    | —              | Սյան ինդեքսը (համարը)։ |
