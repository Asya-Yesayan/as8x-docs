Այս դասը նախատեսված է օգտագործողի էլ․ փոստի տվյալների նկարագրման համար։

## Հատկություններ

| Անվանում | Տվյալների տիպ | Նկարագրություն |
|--------|--------------|---------------|
| Suid | short | Օգտագործողի ներքին համարը (կոդը)։ |
| UserName | short | Օգտագործողի ներքին անունը։ |
| Email | string | Օգտագործողի էլ․ փոստի հասցեն։ |
