## SEQREADURL, SEQAPIKEY

Համակարգում գրանցվող լոգերը ուղարկվում են [Seq](https://datalust.co/) սերվերին, որի հասցեն սահմանվում է [appsettings.json](../../../project/appsettings_json.md) կոնֆիգուրացիոն ֆայլում՝ [Serilog:WriteTo:Args:serverUrl](../../../project/appsettings_json.md#լոգի-գրանցում-seq-սերվերում) դաշտի միջոցով։

**SEQREADURL** պարամետրը նախատեսված է [Seq](https://datalust.co/) սերվերի հասցեն պահպանելու համար։

Առաջադրանքի կատարման ընթացքում գրանցված սխալները հնարավոր է տեսնել «Ադմինիստրատորի աշտ» -> «Առաջադրանքներ» թղթապանակի **«Առաջադրանքների պատմություն»**, **«Առաջադրանքների պատմություն (ըստ քայլերի)»** դիտելու ձևերում՝ [«Կատարման ընթացքում առաջացած սխալներ»](../../../ui/jobs.md#կատարման-ընթացքում-առաջացած-սխալներ) կամ [«Կատարման ընթացքում առաջացած սխալներ Seq-ում»](../../../ui/jobs.md#կատարման-ընթացքում-առաջացած-սխալներ-seq-ում) կոնտեքստային ֆունկցիաների միջոցով։

Նշված կոնտեքստային ֆունկցիաները [Seq](https://datalust.co/) սերվերին միանալու համար նախ ստուգում են **SEQREADURL** պարամետրի արժեքը։ Եթե այն դատարկ է, օգտագործվում է [Serilog:WriteTo:Args:serverUrl](../../../project/appsettings_json.md#լոգի-գրանցում-seq-սերվերում) կոնֆիգուրացիայի արժեքը, իսկ միացման համար կիրառվում է **SEQAPIKEY** պարամետրը։

## 8XJOBNOTIFICATION
