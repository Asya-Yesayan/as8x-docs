# IGrid.ExistsColumn(string) մեթոդ

```c#
public bool ExistsColumn(string columnName);
```

Ստուգում է նշված ներքին անունով սյան առկայությունը ընթացիկ աղյուսակում։ Սյան բացակայության դեպքում վերադարձնում է false:

**Պարամետրեր**

| Անվանում        | Տվյալների տիպ        | Լռությամբ արժեք | Նկարագրություն |
|-----------------|----------------------|----------------|----------------|
|   columnName    |     string           |      -         | Սյան ներքին անունը (կոդը)։ |
