## Փաստաթղթի ստեղծման աղբյուր

Այս enum-ը նախատեսված է փաստաթղթի փաստաթղթի ստեղծման աղբյուրի սահմանման համար։

```c#
public enum DocumentOrigin
{
    Unknown = 0,
    As4xUI = 1,
    As4xScript = 2,
    AsService = 3,
    As8xUI = 4,
    As8xUICode = 5
}
```

**Արժեքների բազմություն**

| Անվանում   | Արժեք | Նկարագրություն                         |
| ---------- | ----- | -------------------------------------- |
| Unknown    | 0     | Փաստաթղթի ստեղծման աղբյուրը անհայտ է։  |
| As4xUI     | 1     | Փաստաթուղթը ստեղծվել է 4X UI-ում։      |
| As4xScript | 2     | Փաստաթուղթը ստեղծվել է 4X սկրիպտում։   |
| AsService  | 3     | Փաստաթուղթը ստեղծվել է 8X սերվիսում։   |
| As8xUI     | 4     | Փաստաթուղթը ստեղծվել է 8X UI-ում։      |
| As8xUICode | 5     | Փաստաթուղթը ստեղծվել է 8X UI-ի կոդում։ |
