Այս enum-ը նախատեսված է հաղորդագրության պատուհանում (MessageBox) լռությամբ կոճակի սահմանման համար։

```c#
public enum MessageBoxDefaultButton
{
    DefaultButton1 = 0,
    DefaultButton2 = 256,
    DefaultButton3 = 512,
    DefaultButton4 = 768
}
```

**Արժեքների բազմություն**

| Անվանում       | Արժեք | Նկարագրություն                                                                 |
| -------------- | ----- | ------------------------------------------------------------------------------ |
| DefaultButton1 | 0     | Լռությամբ կոճակ հանդիսանում է պատուհանում ավելացված կոճակներից ձախից առաջինը։  |
| DefaultButton2 | 256   | Լռությամբ կոճակ հանդիսանում է պատուհանում ավելացված կոճակներից ձախից երկրորդը։ |
| DefaultButton3 | 512   | Լռությամբ կոճակ հանդիսանում է պատուհանում ավելացված կոճակներից ձախից երրորդը։  |
| DefaultButton4 | 768   | Լռությամբ կոճակ հանդիսանում է պատուհանում ավելացված կոճակներից ձախից չորրորդը։ |
