## Հատկություններ

| Անվանում         | Տեսակ                | Նկարագրություն |
|------------------|----------------------|----------------|
| Message      | string                  | Վերադարձնում կամ նշանակում է պրոցեսի ընդհատման պատճառի հաղորդագրությունը։                |
| IsForced          | bool               | Վերադարձնում կամ նշանակում է պրոցեսի հարկադրաբար ընդհատման հայտանիշը։               |
