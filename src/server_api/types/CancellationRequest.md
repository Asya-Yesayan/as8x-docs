## Հատկություններ

| Անվանում         | Տեսակ                | Նկարագրություն |
|------------------|----------------------|----------------|
| Message      | string                  | Վերադարձնում կամ նշանակում է պրոցեսի ընդհատման պատճառի հաղորդագրությունը։                |
| IsForced          | bool               | Վերադարձնում կամ նշանակում է պրոցեսի հարկադրաբար ընդհատման հայտանիշը։ <br> Ընդհատումը համարվում է հարկադրաբար, եթե այն տեղի է ունեցել «Ադմինիստրատորի Աշտ» -> «Միացումների ղեկավարում» -> «Սերվիսային ասինխրոն պրոցեսններ» -> **«Ընդհատել պրոցես(ներ)ը»** կոնտեքստային ֆունկցիայի կամ **api/Process/Stop**, **api/Process/StopAll** api կանչերի միջոցով։             |
