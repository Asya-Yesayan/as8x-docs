# JobNotification enum

## Առաջադրանքի ծանուցման տարբերակը

Այս enum նախատեսված է առաջադրանքի ծանուցման տարբերակի սահմանման համար։ 

```c#
public enum JobNotification
{
    Never = 0,
    OnErrors = 1,
    Always = 2
}
```

**Արժեքների բազմություն**

| Անվանում           | Արժեք | Նկարագրություն          |
| ------------------ | ----- | ------------------------|
| Never              | 0     | Ծանուցումներ չեն ուղարկվում։    |
| OnErrors           | 1     | Ծանուցումներ են ուղարկվում միայն սխալների առաջացման և ձախողման դեպքերում։ |
| Always             | 2     | Ծանուցումներ են ուղարկվում առաջադրանքի կատարման սկզբի, բարեհաջող ավարտի, սխալների առաջացման և ձախողման դեպքերում։ |
