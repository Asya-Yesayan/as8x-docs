# JobStep դաս

## Ներածություն

Այս մոդելը պարունակում է հերթագրված առաջադրանքի կատարման քայլի նկարագրությունը։

## Հատկություններ

| Անվանում         | Տեսակ                | Նկարագրություն |
|------------------|----------------------|----------------|
| ExecutionId      | int                  | Առաջադրանքի id-ն։                |
| JobCode          | string               | Առաջադրանքի կոդը։                |
| Step             | short                | Կատարման քայլի համարը, որոշում է դրա կատարման հերթականությունը բոլոր քայլերի հաջորդականության մեջ։                |
| InterruptOnError | bool                 | Կատարման քայլում սխալ առաջանալու դեպքում առաջադրանքի ամբողջական ընդհատման հայտանիշ։                |
| DPRName          | string               | Կատարման քայլում գործարկվող տվյալների մշակման հարցման (DPR) կոդը։                |
| DPRCaption          | string               | Կատարման քայլում գործարկվող տվյալների մշակման հարցման (DPR) անվանումը։                |
| DPRType          | DPRType              | Կատարման քայլում գործարկվող տվյալների մշակման հարցման (DPR) տեսակի կոդը (առաջադրանքներում կիրառելի են միայն OLAP - 22, JobElement - 23 տեակները):                |
| Params           | string               | Կատարման քայլում գործարկվող տվյալների մշակման հարցման (DPR) պարամետրերի անունները/արժեքները պարունակող json օբյեկտ։ |
| HasError         | bool                 | Առաջադրանքի կատարման ընթացքում սխալների առաջացման հայտանիշ։    |
| NotificationInfo         | [JobStepNotificationInfo](JobStepNotificationInfo.md)     | Առաջադրանքի կատարման քայլի ծանուցման [կարգավորումները](JobStepNotificationInfo.md)։    |
