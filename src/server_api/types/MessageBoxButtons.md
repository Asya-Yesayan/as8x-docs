Այս enum-ը նախատեսված է հաղորդագրության պատուհանում (MessageBox) ավելացվող կոճակների սահմանման համար։

```c#
public enum MessageBoxButtons
{
    OK = 0,
    OKCancel = 1,
    YesNoCancel = 3,
    YesNo = 4
}
```

**Արժեքների բազմություն**

| Անվանում    | Արժեք | Նկարագրություն                                            |
| ----------- | ----- | --------------------------------------------------------- |
| OK          | 0     | Ցուցադրվում է **Լավ** կոճակը։                             |
| OKCancel    | 1     | Ցուցադրվում են **Կատարել**, **Դադարեցնել** կոճակները։     |
| YesNoCancel | 3     | Ցուցադրվում են **Այո**, **Ոչ**, **Դադարեցնել** կոճակները։ |
| YesNo       | 4     | Ցուցադրվում են **Այո**, **Ոչ** կոճակները։                 |
