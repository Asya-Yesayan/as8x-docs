# ASException.Title հատկություն

**Դաս՝** [ASException](../../ASException.md)

```c#
public string Title { get; set; } = string.Empty;
```

Վերադարձնում կամ նշանակում է սխալի պատուհանի գլխագիրը։

Լռությամբ արժեքը դատարկ տող է. այդ դեպքում սխալի պատուհանը ցուցադրվում է լռությամբ գլխագրով։ Արժեքավորվում է [ASException](../Methods/ASException.md) դասի կոնստրուկտորի **title** պարամետրով։
