# ASException.Icon հատկություն

**Դաս՝** [ASException](../../ASException.md)

```c#
public int Icon { get; set; } = -1;
```

Վերադարձնում կամ նշանակում է սխալի պատուհանում ցուցադրվող պատկերի տիպի id-ն։

Լռությամբ արժեքը **-1** է. այդ արժեքի դեպքում սխալի պատուհանը ցույց է տրվում սովորական սխալի պատկերով։ Արժեքավորվում է [ASException](../Methods/ASException.md) դասի կոնստրուկտորի **icon** պարամետրով։
