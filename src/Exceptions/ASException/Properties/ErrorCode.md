# ASException.ErrorCode հատկություն

**Դաս՝** [ASException](../../ASException.md)

```c#
public int ErrorCode { get; set; } = -1;
```

Վերադարձնում կամ նշանակում է սխալի կոդը։

Լռությամբ արժեքը **-1** է. այդ արժեքի դեպքում սխալին տրված է ընդհանուր (չտեսակավորված) կարգավիճակ։ Արժեքավորվում է [ASException](../Methods/ASException.md) դասի կոնստրուկտորի **code** պարամետրով։ Կանչող կոդը այս արժեքի միջոցով կարող է որոշել, թե ինչպես արձագանքել կոնկրետ սխալին (օրինակ՝ լուռ բաց թողնել, ցուցադրել, փորձել կրկին)։
