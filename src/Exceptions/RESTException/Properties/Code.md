# RESTException.Code հատկություն

**Դաս՝** [RESTException](../../RESTException.md)

```c#
public string Code { get; set; }
```

Վերադարձնում կամ նշանակում է սխալի կոդը։

Արժեքավորվում է [RESTException](../Methods/RESTException.md) դասի կոնստրուկտորի **code** պարամետրով։ Սերվերից պատասխանին փոխանցվում է client-ին և հասանելի է դառնում [ApiException](../../ApiException.md).[Code](../../ApiException/Properties/Code.md) հատկության միջոցով։
