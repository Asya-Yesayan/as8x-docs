# RESTException.Title հատկություն

**Դաս՝** [RESTException](../../RESTException.md)

```c#
public string Title { get; set; }
```

Վերադարձնում կամ նշանակում է սխալի պատուհանի գլխագիրը։

Արժեքավորվում է [RESTException](../Methods/RESTException.md) դասի կոնստրուկտորի **title** պարամետրով։ Սերվերից փոխանցվում է client-ին և օգտագործվում է UI-ում սխալի պատուհանի վերնագիրը ձևավորելու համար։
