# ApiException.InnerMessage հատկություն

**Դաս՝** [ApiException](../../ApiException.md)

```c#
public string InnerMessage { get; set; } = null;
```

Վերադարձնում կամ նշանակում է ներքին սխալի լրացուցիչ հաղորդագրությունը։

Լռությամբ արժեքը **null** է. այդ դեպքում սխալին լրացուցիչ հաղորդագրություն կցված չէ։ Ոչ null արժեքի դեպքում օգտագործվում է սերվերից ստացված ներքին սխալի մասին լրացուցիչ տեղեկատվությունը պահպանելու համար (օրինակ՝ սխալի հիմքում ընկած ենթահամակարգի հաղորդագրությունը)։
