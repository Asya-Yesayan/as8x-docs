# ApiException.Report հատկություն

**Դաս՝** [ApiException](../../ApiException.md)

```c#
public StorageInfo Report { get; set; }
```

Վերադարձնում կամ նշանակում է տեքստային հաշվետվության պահեստավորման տվյալները։

Արժեքավորվում է, երբ ApiException-ը ձևավորվում է սերվերի [DocumentException](../../DocumentException.md)-ից՝ **Report** տիպով։ UI-ն այս տվյալները կարող է օգտագործել՝ համապատասխան տեքստային հաշվետվությունը բացելու համար։
