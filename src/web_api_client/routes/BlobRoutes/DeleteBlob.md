```c#
public bool DeleteBlob(string container, 
                       string blobName)
```

Հեռացնում է ֆայլը պահոցից ըստ անվան և կոնտեյների։ 

**Պարամետրեր**

* `container` - Կոնտեյների անունը։ 
* `blobName` - Ֆայլի անունը` ներառյալ ֆայլի ընդլայնումը։
