# DialogWindow.StoreValuesIsChecked հատկություն

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public bool StoreValuesIsChecked { get; }
```

Վերադարձնում է **true**, եթե երկխոսության պատուհանի արժեքների պահպանման ռեժիմը միացված է (օգտագործողը նշել է երկխոսության պատուհանի վերնագրի վահանակում գտնվող համապատասխան կոճակը)։
