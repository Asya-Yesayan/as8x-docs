# DialogWindow.HelpId հատկություն

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public string HelpId { get; set; }
```

Վերադարձնում կամ նշանակում է երկխոսության պատուհանի օգնության նույնականացուցիչը (օգտագործվում է **F1** ստեղնի համար)։

## Օրինակ

```c#
// նշանակում է երկխոսության պատուհանի օգնության նույնականացուցիչը
this.HelpId = "CurrencyFilterDialog";
```
