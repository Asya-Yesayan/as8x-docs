# DialogWindow.SetFocusOnActivation հատկություն

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public Control SetFocusOnActivation { get; set; }
```

Վերադարձնում կամ նշանակում է այն դաշտը, որի վրա կտեղափոխվի մուտքագրման ֆոկուսը երկխոսության պատուհանի առաջին ակտիվացման ժամանակ։ Լռությամբ ֆոկուսը տեղափոխվում է երկխոսության պատուհանին առաջինը ավելացված դաշտին, սակայն ծրագրավորողը կարող է անձամբ սահմանել այլ դաշտ։
