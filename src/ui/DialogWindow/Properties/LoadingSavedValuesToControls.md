# DialogWindow.LoadingSavedValuesToControls հատկություն

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public bool LoadingSavedValuesToControls { get; private set; }
```

Վերադարձնում է **true**, եթե ընթացիկ պահին երկխոսության պատուհանի դաշտերում բեռնվում են պահպանված արժեքները։ Հատկությունը կարող է օգտագործվել ժառանգորդ դասերի կողմից՝ բեռնման ընթացքում խուսափելու այն գործողություններից, որոնք նախատեսված են միայն օգտագործողի կողմից արժեքի ձեռքով փոփոխման համար։
