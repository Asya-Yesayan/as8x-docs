# DialogWindow.Mode հատկություն

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public DialogWindowMode Mode { get; set; }
```

Վերադարձնում կամ նշանակում է երկխոսության պատուհանի բացման ռեժիմը՝ նոր գրառման ստեղծում, խմբագրում, դիտում կամ հեռացում։

## Օրինակ

```c#
// նշանակում է երկխոսության պատուհանի ռեժիմը՝ նոր գրառման ստեղծում
this.Mode = DialogWindowMode.NewMode;
```
