# DialogWindow.Mode հատկություն

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public DialogWindowMode Mode { get; set; }
```

Վերադարձնում կամ նշանակում է երկխոսության պատուհանի աշխատանքի ռեժիմը՝ նոր գրառման ստեղծում, խմբագրում, դիտում կամ ջնջում։ Ռեժիմի փոփոխման ժամանակ ավտոմատ կերպով թարմացվում են պատուհանի վերնագիրը և **OK**/**Cancel**/**Next** կոճակների բովանդակությունն ու տեսանելիությունը։

**Արժեք**

**DialogWindowMode** տիպի արժեք.

| Արժեք | Նկարագրություն |
|-------|----------------|
| None | Հատուկ ռեժիմ սահմանված չէ։ |
| NewMode | Նոր գրառման ստեղծման ռեժիմ։ |
| NewModeWithNext | Նոր գրառման ստեղծման ռեժիմ՝ **Next** կոճակի աջակցությամբ։ |
| EditMode | Գոյություն ունեցող գրառման խմբագրման ռեժիմ։ |
| ViewMode | Միայն դիտման (կարդալու) ռեժիմ։ |
| DeleteMode | Ջնջման հաստատման ռեժիմ։ |
