# DialogWindow.Item[string] հատկություն

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public object this[string name] { get; set; }
```

Վերադարձնում կամ նշանակում է երկխոսության պատուհանի՝ տրված ներքին անունով դաշտի արժեքը։ Դաշտի տիպից կախված՝ վերադարձվում կամ ստացվում է համապատասխան տիպի արժեք (օրինակ՝ տեքստ, ամսաթիվ, թիվ, տրամաբանական արժեք և այլն)։ Կարդալու ժամանակ, եթե տրված անունով դաշտ չկա, վերադարձվում է **null**։

**Արժեք**

Տրված ներքին անունով դաշտի արժեքը՝ **object** տիպով։

**Բացառություններ**

| Տիպ | Նկարագրություն |
|-----|----------------|
| InvalidOperationException | Առաջանում է նշանակելու ժամանակ, եթե տրված ներքին անունով դաշտ գոյություն չունի։ |
