# DialogWindow.Caption հատկություն

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public string Caption { get; set; }
```

Վերադարձնում կամ նշանակում է երկխոսության պատուհանի գլխագիրը։ 

## Օրինակ

```c#
// նշանակում է երկխոսության պատուհանի գլխագիրը
this.Caption = "Ֆիլտրի պարամետրեր";

// ստանում է ընթացիկ գլխագիրը
string title = this.Caption;
```
