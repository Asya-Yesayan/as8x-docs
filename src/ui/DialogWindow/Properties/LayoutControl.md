# DialogWindow.LayoutControl հատկություն

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public LayoutControl LayoutControl;
```

Վերադարձնում է երկխոսության պատուհանի դաշտերը պարունակող դասավորման հիմնական տարրը (**LayoutControl**)։ Բոլոր դաշտերն ու դասավորման խմբերը, որոնք ավելացվում են առանց **container** պարամետրի, տեղադրվում են այս տարրի մեջ։
