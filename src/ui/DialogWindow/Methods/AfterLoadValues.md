# DialogWindow.AfterLoadValues() մեթոդ

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public virtual void AfterLoadValues();
```

Կանչվում է երկխոսության պատուհանի դաշտերի արժեքների բեռնման ավարտից հետո։ Ընդլայնման կետ ժառանգ դասերի համար, որպեսզի իրականացնեն բեռնման հետևանքով անհրաժեշտ լրացուցիչ գործողություններ։ Բազային իրականացումը գործողություն չի կատարում։
