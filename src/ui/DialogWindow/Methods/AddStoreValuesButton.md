# DialogWindow.AddStoreValuesButton() մեթոդ

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public void AddStoreValuesButton();
```

Երկխոսության պատուհանի վերնագրի վահանակում ավելացնում է երեք կոճակ՝ արժեքների պահպանման միացում/անջատման կոճակ, պահպանվող դաշտերի ընտրման կարգավորումների կոճակ և պահպանված արժեքների մաքրման կոճակ։ Հասանելի է **Ctrl+S** (պահպանման միացում/անջատում) և **Ctrl+D** (մաքրում) ստեղների համակցություններից։

**Բացառություններ**

| Տիպ | Նկարագրություն |
|-----|----------------|
| InvalidOperationException | Առաջանում է, եթե երկխոսության պատուհանի ներքին անունը (**Name**) սահմանված չէ։ |
