# DialogWindow.CheckControlExistence(string) մեթոդ

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
protected internal void CheckControlExistence(string name);
```

Ստուգում է, արդյոք երկխոսության պատուհանում տրված ներքին անունով դաշտ արդեն գոյություն ունի։ Այո դեպքում նետում է բացառություն։ Կանչվում է բոլոր **Add*** մեթոդների ներսից՝ կրկնող անունների կանխարգելման համար։

**Պարամետրեր**

| Անվանում | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն |
|----------|--------------|----------------|----------------|
| name | string | — | Ստուգվող դաշտի ներքին անունը։ |

**Բացառություններ**

| Տիպ | Նկարագրություն |
|-----|----------------|
| InvalidOperationException | Առաջանում է, եթե տրված անունով դաշտ արդեն գոյություն ունի։ |
