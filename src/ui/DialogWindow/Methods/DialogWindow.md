# DialogWindow.DialogWindow(Window, bool, bool) մեթոդ

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public DialogWindow(Window owner = null, bool canResize = false, bool configureModal = true);
```

Ստեղծում է **DialogWindow** դասի նոր օրինակ՝ տրված ծնող պատուհանով և ձևավորման կարգավորումներով։ Կառուցում է երկխոսության պատուհանի հիմնական դասավորման տարրը, **OK**, **Cancel**, **Next** կոճակները և կարգավիճակի տողը։ **configureModal** պարամետրը որոշում է, արդյոք պատուհանը ստեղծվում է որպես մոդալ (ընդհատող), թե ոչ։

**Պարամետրեր**

| Անվանում | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն |
|----------|--------------|----------------|----------------|
| owner | Window | null | Երկխոսության պատուհանի ծնող պատուհանը։ **null** դեպքում օգտագործվում է հոսթինգային միջավայրի հիմնական պատուհանը։ |
| canResize | bool | false | **true** արժեքի դեպքում երկխոսության պատուհանի չափերը կարող են փոփոխվել, հակառակ դեպքում՝ ոչ։ |
| configureModal | bool | true | **true** արժեքի դեպքում երկխոսության պատուհանը կարգավորվում է որպես մոդալ (ընդհատող), հակառակ դեպքում՝ որպես ոչ մոդալ։ |
