# DialogWindow.OnStoreDefaultValues() մեթոդ

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public virtual void OnStoreDefaultValues();
```

Երկխոսության պատուհանի դաշտերի ընթացիկ արժեքները ֆիքսում է որպես կանխադիր արժեքներ՝ **StoredValues** ցուցակի **DefaultValue** դաշտերում։ Կիրառվում է երկխոսության պատուհանի բեռնման ժամանակ՝ դաշտերի կանխադիր արժեքներն հիշելու համար, որպեսզի հետագայում օգտագործողը կարողանա մաքրման կոճակի օգնությամբ վերականգնել դրանք։
