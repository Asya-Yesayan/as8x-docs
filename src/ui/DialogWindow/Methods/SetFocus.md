# DialogWindow.SetFocus(Control, bool) մեթոդ

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
protected static void SetFocus(Control focusableControl, bool selectAll = true);
```

Տրված դաշտի վրա սահմանում է մուտքագրման ֆոկուսը։ Կախված դաշտի տիպից՝ կատարվում է ֆոկուսի անցնում համապատասխան ներքին տարրին (օրինակ՝ կոդի մուտքագրման դաշտին, ամսաթվային մուտքագրման դաշտին և այլն)։ Եթե դաշտը գտնվում է էջի (tab) մեջ, ապա համապատասխան էջն ավտոմատ կերպով ակտիվացվում է։

**Պարամետրեր**

| Անվանում | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն |
|----------|--------------|----------------|----------------|
| focusableControl | Control | — | Դաշտը, որի վրա պետք է սահմանվի ֆոկուսը։ |
| selectAll | bool | true | **true** արժեքի դեպքում դաշտի ընթացիկ տեքստն ընտրվում է ամբողջությամբ։ |
