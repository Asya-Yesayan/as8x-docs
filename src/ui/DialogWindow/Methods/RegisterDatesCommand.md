# DialogWindow.RegisterDatesCommand() մեթոդ

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public static void RegisterDatesCommand();
```

Գրանցում է **Alt+D** shortcut-ի միջոցով **DateEditControl** դաշտերի համար, որպեսզի այն ակտիվացվի և ցուցադրվի հատուկ ամսաթվերի popup տեսակի պատուհանը։ Մեթոդը ապահով է բազմակի կանչերի համար. եթե հրամանն արդեն գրանցված է, կրկնակի գրանցում չի կատարվում։
