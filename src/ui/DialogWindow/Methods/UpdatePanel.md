# DialogWindow.UpdatePanel(object, bool) մեթոդ

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public void UpdatePanel(object key, bool isRefresh = false);
```

Թարմացնում է երկխոսության պատուհանի հետ կապված վահանակը (ցուցակային կամ ծառային)։ **isRefresh** **true** արժեքի դեպքում կատարվում է վահանակի ամբողջական թարմացում, հակառակ դեպքում՝ թարմացվում է միայն տրված բանալիով տարրը։ Եթե **PanelId** սահմանված չէ, մեթոդը գործողություն չի կատարում։

**Պարամետրեր**

| Անվանում | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն |
|----------|--------------|----------------|----------------|
| key | object | — | Թարմացվող տարրի բանալին։ |
| isRefresh | bool | false | **true** արժեքի դեպքում կատարվում է վահանակի ամբողջական թարմացում, **false** արժեքի դեպքում՝ տրված բանալիով տարրի մասնակի թարմացում։ |

## Օրինակ

```c#
// թարմացնում է վահանակում տրված բանալիով տարրը
UpdatePanel(documentId);

// կատարում է վահանակի ամբողջական թարմացում
UpdatePanel(null, isRefresh: true);
```
