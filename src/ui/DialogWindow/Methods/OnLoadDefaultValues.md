# DialogWindow.OnLoadDefaultValues() մեթոդ

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
public virtual void OnLoadDefaultValues();
```

Բեռնում է երկխոսության պատուհանի դաշտերի կանխադիր արժեքները՝ պահպանվող արժեքների ցուցակում գրառված **DefaultValue**-ից։ Կանչվում է, երբ օգտագործողը սեղմում է "մաքրման" կոճակը երկխոսության պատուհանի վերնագրի վահանակում (կամ **Ctrl+D** shortcut-ի միջոցով)։
