# DialogWindow.GetDataSourceName() մեթոդ

## Նկարագիր

**Դաս՝** [DialogWindow](../DialogWindow.md)

```c#
protected virtual string GetDataSourceName();
```

Վերադարձնում է երկխոսության պատուհանի հետ կապված տվյալների աղբյուրի (DataSource) համակարգային անունը, կամ **null**, եթե այդպիսին չկա։ Ընդլայնման կետ ժառանգորդ դասերի համար։ Բազային իրականացումը վերադարձնում է **null**։

**Վերադարձնում է**

Տվյալների աղբյուրի համակարգային անունը, կամ **null**՝ եթե երկխոսության պատուհանը չունի կապված տվյալների աղբյուր։
