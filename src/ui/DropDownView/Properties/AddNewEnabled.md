# DropDownView.AddNewEnabled հատկություն

## Նկարագիր

**Դաս՝** [DropDownView](../DropDownView.md)

```c#
public virtual bool AddNewEnabled { get; }
```

Սահմանում է բացվող ցուցակի դիտելու ձևից «Ավելացնել» գործողության հասանելիության իրավասությունը՝ [AddNewSupported](AddNewSupported.md) հատկության հետ համատեղ։

Հատկությունը չմշակելու դեպքում վերադարձնում է [AddNewSupported](AddNewSupported.md) հատկության արժեքը։
