# DropDownView.DropDownView() մեթոդ

## Նկարագիր

**Դաս՝** [DropDownView](../DropDownView.md)

```c#
public DropDownView()
```

Ստեղծում է DropDownView դասի օբյեկտ և կոնստրուկտորում ստուգում է DropDownView ատրիբուտի առկայությունը։

DropDownView ատրիբուտի առկայության դեպքում արժեքավորվում են Name, Caption և SupportsExtensions հատկությունները՝ ատրիբուտում նշված տվյալներով։ CodeDictionary հատկությունը նախնական արժեքավորվում է null-ով։

DropDownView ատրիբուտի բացակայության դեպքում name դաշտը արժեքավորվում է null-ով, supportsExtensions դաշտը՝ false-ով։
