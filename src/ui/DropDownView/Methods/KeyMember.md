# DropDownView.KeyMember(int index) մեթոդ

## Նկարագիր

**Դաս՝** [DropDownView](../DropDownView.md)

```c#
public virtual object KeyMember(int index)
```

Սահմանում է բացվող ցուցակի դիտելու ձևի՝ տրված ինդեքսով տողի բանալու արժեքը։

Մեթոդը չմշակելու դեպքում առաջանում է **NotImplementedException** տիպի սխալ։

**Պարամետրեր**

| Անվանում | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն |
| --- | --- | --- | --- |
| index | int | - | Տողի ինդեքսը բացվող ցուցակի դիտելու ձևի տողերի ցուցակում։ |

**Վերադարձնում է**

Տրված ինդեքսով տողի բանալի արժեքը։
