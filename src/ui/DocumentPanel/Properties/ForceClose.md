# DocumentPanel&lt;T&gt;.ForceClose հատկություն
## Նկարագիր
**Դաս՝** [DocumentPanel](../DocumentPanel.md)
```c#
protected bool ForceClose { get; set; } = false;
```
Վերադարձնում կամ նշանակում է փաստաթղթի պատուհանը հարկադիր փակելու դրոշակը։ Հատկության **true** արժեքի դեպքում պատուհանը փակվում է առանց փոփոխությունների պահպանման հարցման։ Լռությամբ արժեքը՝ **false**։
