# DocumentPanel&lt;T&gt;.IsLoadingDocumentToControls հատկություն

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

```c#
public bool IsLoadingDocumentToControls { get; }
```

Վերադարձնում է **true**, եթե տվյալ պահին իրականացվում է փաստաթղթի արժեքների բեռնումը դաշտերի մեջ։ Հատկությունը կարող է օգտագործվել արժեքի փոփոխման իրադարձության մշակիչներում՝ բեռնման ընթացքում առաջացող իրադարձությունները բացառելու համար։
