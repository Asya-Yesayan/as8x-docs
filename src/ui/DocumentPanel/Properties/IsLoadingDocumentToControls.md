# DocumentPanel&lt;T&gt;.IsLoadingDocumentToControls հատկություն
## Նկարագիր
**Դաս՝** [DocumentPanel](../DocumentPanel.md)
```c#
public bool IsLoadingDocumentToControls { get; }
```
Վերադարձնում է **true**, եթե ընթացքում է փաստաթղթի արժեքների բեռնումը դաշտերի մեջ։ Հատկության արժեքը կարող է օգտագործվել արժեքի փոփոխման իրադարձության մշակիչներում՝ բեռնման ընթացքում առաջացած իրադարձությունները բացառելու համար։
