# DocumentPanel&lt;T&gt;.SkipReadOnlyControlsValidation հատկություն

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

```c#
protected virtual bool SkipReadOnlyControlsValidation { get; } = true;
```

Սահմանում է վալիդացման ընթացքում միայն կարդալու վիճակում գտնվող դաշտերը բացառելու հայտանիշը։ Հատկության լռությամբ արժեքը՝ **true**, որի դեպքում միայն կարդալու դաշտերի վալիդացիան բաց է թողնվում։
