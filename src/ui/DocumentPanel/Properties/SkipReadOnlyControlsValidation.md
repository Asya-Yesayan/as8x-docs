# DocumentPanel&lt;T&gt;.SkipReadOnlyControlsValidation հատկություն

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

```c#
protected virtual bool SkipReadOnlyControlsValidation { get; } = true;
```

Սահմանում է վալիդացման ընթացքում readOnly դաշտերը բացառելու հայտանիշը։ Հատկության լռությամբ արժեքը true է, որի դեպքում readOnly դաշտերի վալիդացիան բաց է թողնվում։
