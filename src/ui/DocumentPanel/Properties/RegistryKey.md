# DocumentPanel&lt;T&gt;.RegistryKey հատկություն

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

```c#
protected string RegistryKey { get; set; } = "";
```

Վերադարձնում կամ նշանակում է registry-ում փաստաթղթի վահանակի կարգավորումները պահելու ենթաբանալին։ Լռությամբ արժեքը՝ դատարկ տող, որի դեպքում կարգավորումները չեն պահպանվում։
