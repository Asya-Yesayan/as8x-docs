# DocumentPanel&lt;T&gt;.ShowTitleDetails հատկություն

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

```c#
protected virtual bool ShowTitleDetails { get; }
```

Սահմանում է փաստաթղթի պատուհանի վերնագրում ISN-ի և վիճակի ցուցադրման դրոշակը։ Հատկության լռությամբ արժեքը՝ **true**, որի դեպքում վերնագիրը պարունակում է փաստաթղթի ISN-ը և ընթացիկ վիճակը (նոր, դիտում, խմբագրում)։
