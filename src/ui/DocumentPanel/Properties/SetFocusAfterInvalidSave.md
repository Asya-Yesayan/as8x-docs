# DocumentPanel&lt;T&gt;.SetFocusAfterInvalidSave հատկություն

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

Վերադարձնում կամ նշանակում է անհաջող պահպանումից հետո ֆոկուսը տեղափոխվող դաշտը։ Հատկության արժեքը թարմացվում է **InvalidFieldValue** տիպի սխալի դեպքում՝ պարունակելով այն դաշտը, որի արժեքը չի անցել վալիդացիան։
