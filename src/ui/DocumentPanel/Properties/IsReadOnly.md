# DocumentPanel&lt;T&gt;.IsReadOnly հատկություն

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

```c#
protected bool IsReadOnly { get; set; }
```

Վերադարձնում կամ նշանակում է փաստաթղթի վահանակի readOnly վիճակով բացվելու հայտանիշը։ Հատկության արժեքը նշանակվում է վահանակի ցուցադրման սկզբում՝ ըստ բացման ռեժիմի (խմբագրման կամ դիտման)։
