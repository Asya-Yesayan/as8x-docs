# DocumentPanel&lt;T&gt;.AllowExportToJSON() մեթոդ

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

```c#
protected virtual bool AllowExportToJSON()
```

Սահմանում է, արդյոք թույլատրված է փաստաթղթի **JSON** ֆորմատով արտահանումը։
Մեթոդը չմշակելու դեպքում վերադարձնում է **true**։

**Վերադարձնում է**

**true** արժեքի դեպքում **JSON** ֆորմատով արտահանման կոճակը հասանելի է, **false** արժեքի դեպքում՝ թաքնված։
