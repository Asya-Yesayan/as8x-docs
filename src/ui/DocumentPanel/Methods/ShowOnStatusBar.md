# DocumentPanel&lt;T&gt;.ShowOnStatusBar(GridControl) մեթոդ

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

```c#
public void ShowOnStatusBar(GridControl grid)
```

Տրված աղյուսակի ընթացիկ տողի և ընտրված տողերի թիվը ցուցադրում է պատուհանի status bar-ում։ Աղյուսակի կամ դրա ներկայացման բացակայության դեպքում մեթոդը որևէ գործողություն չի կատարում։

| Անվանում | Տվյալների տիպ | Լռությամբ արժեք | Նկարագրություն |
| --- | --- | --- | --- |
| grid | GridControl | - | Աղյուսակը, որի վիճակը ցուցադրվում է status bar-ում։ |
