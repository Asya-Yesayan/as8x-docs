# DocumentPanel&lt;T&gt;.AllowPrint() մեթոդ

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

```c#
protected bool AllowPrint()
```

Վերադարձնում է **true**, եթե թույլատրված է փաստաթղթի տպումը։

**Վերադարձնում է**

**true** արժեքի դեպքում փաստաթղթի տպումը թույլատրված է, **false** արժեքի դեպքում՝ արգելված։
