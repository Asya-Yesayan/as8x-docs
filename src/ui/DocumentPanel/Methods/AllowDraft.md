# DocumentPanel&lt;T&gt;.AllowDraft() մեթոդ

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

```c#
protected virtual bool AllowDraft()
```

Սահմանում է, արդյոք փաստաթղթի համար թույլատրված է **«Պահպանել որպես սևագիր»** հնարավորությունը։
Մեթոդը չմշակելու դեպքում վերադարձնում է փաստաթղթի սխեմայի **DisableDraft** հայտանիշի հակառակ արժեքը։ **false** արժեքի դեպքում **«Պահպանել որպես սևագիր»** կոճակը թաքնված է։ **true** արժեքի դեպքում խմբագրման ռեժիմում կոճակը մնում է անհասանելի, իսկ նոր փաստաթղթի ռեժիմում՝ հասանելի է անմիջապես։

**Վերադարձնում է**

**true** արժեքի դեպքում **«Պահպանել որպես սևագիր»** կոճակը հասանելի է, **false** արժեքի դեպքում՝ թաքնված։
