# DocumentPanel&lt;T&gt;.AllowDraft() մեթոդ

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

```c#
protected virtual bool AllowDraft()
```

Սահմանում է, արդյոք փաստաթղթի համար թույլատրված է **«Պահպանել որպես սևագիր»** հնարավորությունը։

Մեթոդը չմշակելու դեպքում վերադարձնում է փաստաթղթի սխեմայի **DisableDraft** հայտանիշի հակառակ արժեքը։ Հատկության **false** արժեքի դեպքում **«Պահպանել որպես սևագիր»** կոճակը թաքնված է, հակառակ դեպքում հասանելի է միայն նոր փաստաթղթի ռեժիմում։
