# DocumentPanel&lt;T&gt;.Validate() մեթոդ
## Նկարագիր
**Դաս՝** [DocumentPanel](../DocumentPanel.md)
```c#
protected virtual bool Validate()
```
Սահմանում է փաստաթղթի դաշտերի վալիդացման գործողությունները։

**Վերադարձնում է**

**true** արժեքի դեպքում բոլոր դաշտերը վալիդ են, **false** արժեքի դեպքում՝ առկա է վալիդացման սխալով դաշտ։
