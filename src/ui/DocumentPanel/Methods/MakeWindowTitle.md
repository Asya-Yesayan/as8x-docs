# DocumentPanel&lt;T&gt;.MakeWindowTitle() մեթոդ

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

```c#
protected void MakeWindowTitle()
```

Թարմացնում է փաստաթղթի պատուհանի վերնագիրը և տպման կոճակների վիճակը։ Կարգավորում է **JSON** արտահանման կոճակի հասանելիությունը՝ ըստ փաստաթղթի նոր լինելու, և տպման կոճակների ([AllowPrint](AllowPrint.md) մեթոդի **true** արժեքի դեպքում) հասանելիությունը՝ ըստ [AllowPrintOnNew](AllowPrintOnNew.md) մեթոդի և փաստաթղթի նոր լինելու արժեքների։
