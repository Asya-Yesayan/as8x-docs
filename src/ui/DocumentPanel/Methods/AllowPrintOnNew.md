# DocumentPanel&lt;T&gt;.AllowPrintOnNew() մեթոդ

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

```c#
protected virtual bool AllowPrintOnNew()
```

Սահմանում է, արդյոք թույլատրված է նոր փաստաթղթի տպումը պահպանումից առաջ։
Մեթոդը չմշակելու դեպքում վերադարձնում է **false**։

**Վերադարձնում է**

**true** արժեքի դեպքում նոր փաստաթղթի տպումը թույլատրված է, **false** արժեքի դեպքում՝ արգելված մինչև առաջին պահպանումը։
