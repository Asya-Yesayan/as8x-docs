# DocumentPanel&lt;T&gt;.AllowSaveButton() մեթոդ

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

```c#
protected virtual bool AllowSaveButton()
```

Սահմանում է, արդյոք **«Պահպանել»** կոճակը հասանելի է փաստաթղթի վահանակում։
Մեթոդը չմշակելու դեպքում վերադարձնում է **true**։ Խմբագրման ռեժիմում **«Պահպանել»** կոճակը հասանելի է դառնում փոփոխությունների առկայության դեպքում կամ վահանակի ցուցադրումից անմիջապես, եթե այն բացվել է **buttonsSaveEnabled** պարամետրի **true** արժեքով։ Նոր փաստաթղթի ռեժիմում կոճակը հասանելի է անմիջապես։ [AllowSave](AllowSave.md) մեթոդի **false** արժեքի դեպքում կոճակը մնում է անհասանելի։

**Վերադարձնում է**

**true** արժեքի դեպքում **«Պահպանել»** կոճակը հասանելի է, **false** արժեքի դեպքում՝ թաքնված։
