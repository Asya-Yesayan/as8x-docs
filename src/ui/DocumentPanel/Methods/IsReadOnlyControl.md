# DocumentPanel&lt;T&gt;.IsReadOnlyControl(Control) մեթոդ

## Նկարագիր

**Դաս՝** [DocumentPanel](../DocumentPanel.md)

Վերադարձնում է **true**, եթե տրված դաշտը գտնվում է readOnly վիճակում։
