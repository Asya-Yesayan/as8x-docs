# TreeEdit.Definition հատկություն

## Նկարագիր

**Դաս՝** [TreeEdit](../TreeEdit.md)

```c#
public TreeDefinition Definition { get; internal set; }
```

Վերադարձնում է ծառի սահմանումը (**TreeDefinition**)՝ ConfigureFromDefinition մեթոդին փոխանցված ներքին անվամբ։ Սահմանումը պարունակում է ծառի կառուցվածքային նկարագրությունը, ինչպիսիք են կոդի երկարությունը, թույլատրված գործողությունները և կից փաստաթղթերի ցանկը։
