# TreeEdit.AllowHistory հատկություն

## Նկարագիր

**Դաս՝** [TreeEdit](../TreeEdit.md)

```c#
public virtual bool AllowHistory { get; internal set; }
```

Սահմանում է ծառի ընթացիկ տողում պարունակվող փաստաթղթի պատմությունը դիտելու իրավասությունը։ Հատկությունը արժեքավորվում է ConfigureFromDefinition մեթոդի կանչի արդյունքում՝ ծառի նկարագրության (TreeDefinition) AllowHistory հատկության հիման վրա։

Հատկության true արժեքի դեպքում ծառ-տեղեկատուի կոնտեքստային մենյուում ցուցադրվում է **«Պատմություն»** կոնտեքստային ֆունկցիան, որի կատարման արդյունքում բացվում է ընթացիկ տողում պարունակվող փաստաթղթի փոփոխությունների պատմությունը։
