# TreeEdit.Panel հատկություն

## Նկարագիր

**Դաս՝** [TreeEdit](../TreeEdit.md)

```c#
public TreeEditPanel Panel { get; private set; }
```

Վերադարձնում է ծառի խմբագրման էջի վահանակը (**TreeEditPanel**)։ Հատկությունը արժեքավորվում է [Show](../Methods/Show.md) մեթոդի կանչի արդյունքում, երբ վահանակը ստեղծվում և տեղադրվում է պատուհանում։
