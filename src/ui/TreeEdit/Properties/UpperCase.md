# TreeEdit.UpperCase հատկություն

## Նկարագիր

**Դաս՝** [TreeEdit](../TreeEdit.md)

```c#
public virtual bool UpperCase { get; internal set; }
```

Սահմանում է ծառի տողի կոդի մեծատառերով պահպանման իրավասությունը։ Հատկության լռությամբ արժեքը false է։

Հատկության true արժեքի դեպքում ծառի տողի կոդը պահպանվում է մեծատառերով՝ անկախ օգտագործողի մուտքագրումից։
