# ViewExtension նկարագրություն

## Ներածություն

Այս դասը նախատեսված է դիտելու ձևի ընդլայնման նկարագրման համար։ 

Ընդլայնման միջոցով հնարավոր է ավելացնել հաշվարկվող սյուներ, սահմանել ֆիլտրացում, խմբավորում, պայմանական ձևավորում և այլ լրացուցիչ կարգավորումներ դիտելու ձևի համար։

## Հատկություններ

| Անվանում | Նկարագրություն |
|----------|----------------|
| [ArmenianCaption](Properties/ArmenianCaption.md) | Վերադարձնում կամ նշանակում է ընդլայնման հայերեն վերնագիրը։ |
| [CalculatedExtenderParamsValues](Properties/CalculatedExtenderParamsValues.md) | Վերադարձնում կամ նշանակում է ընդլայնման հաշվարկված պարամետրերի արժեքների Dictionary-ն։ |
| [CalculationMode](Properties/CalculationMode.md) | Վերադարձնում կամ նշանակում է դիտելու ձևի ընդլայնման ձևավորման ռեժիմը։ |
| [ColumnOperations](Properties/ColumnOperations.md) | Վերադարձնում կամ նշանակում է ընդլայնման սյուների գործողությունների ցուցակը։ |
| [ConditionalFormatting](Properties/ConditionalFormatting.md) | Վերադարձնում կամ նշանակում է ընդլայնման պայմանական ձևավորման տողը։ |
| [DynamicHandler](Properties/DynamicHandler.md) | Վերադարձնում է ընդլայնման դինամիկ մշակիչը։ |
| [EnableRowProcessing](Properties/EnableRowProcessing.md) | Սահմանում է ընդլայնման տողերի մշակման իրավասությունը։ |
| [EnglishCaption](Properties/EnglishCaption.md) | Վերադարձնում կամ նշանակում է ընդլայնման անգլերեն վերնագիրը։ |
| [ExtenderParamsInitialValues](Properties/ExtenderParamsInitialValues.md) | Վերադարձնում կամ նշանակում է ընդլայնման պարամետրերի սկզբնական արժեքների Dictionary-ն։ |
| [ExtenderSchema](Properties/ExtenderSchema.md) | Վերադարձնում կամ նշանակում է ընդլայնման սխեման։ |
| [FilterString](Properties/FilterString.md) | Վերադարձնում կամ նշանակում է ընդլայնման ֆիլտրի պայմանը։ |
| [GroupingString](Properties/GroupingString.md) | Վերադարձնում կամ նշանակում է ընդլայնման խմբավորման պայմանը։ |
| [Name](Properties/Name.md) | Վերադարձնում կամ նշանակում է ընդլայնման ներքին անունը (կոդ)։ |
| [View](Properties/View.md) | Վերադարձնում է ընդլայնմանը կցված դիտելու ձևը։ |

| [ProcessRow(ExtendableRow)](Methods/ProcessRow.md) | Սահմանում է ընդլայնման՝ յուրաքանչյուր տողի մշակման գործողությունը։ |
