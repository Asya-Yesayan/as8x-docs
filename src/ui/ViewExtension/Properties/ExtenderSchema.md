# ViewExtension.ExtenderSchema հատկություն

## Նկարագիր

**Դաս՝** [ViewExtension](../ViewExtension.md)

```c#
public ExtenderSchemaEx ExtenderSchema { get; set; }
```

Վերադարձնում կամ նշանակում է ընդլայնման սխեման։ Սխեման պարունակում է ընդլայնմանը կցված տվյալների աղբյուրի ընդլայնիչի սյուների և պարամետրերի նկարագրությունները, ինչպես նաև հայերեն և անգլերեն վերնագրերը։
