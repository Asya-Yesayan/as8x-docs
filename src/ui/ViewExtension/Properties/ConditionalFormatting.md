# ViewExtension.ConditionalFormatting հատկություն

## Նկարագիր

**Դաս՝** [ViewExtension](../ViewExtension.md)

```c#
public string ConditionalFormatting { get; set; }
```

Վերադարձնում կամ նշանակում է ընդլայնման պայմանական ձևավորման պայմանը։ Պայմանական ձևավորման միջոցով սահմանվում է, թե դիտելու ձևի տողերի և սյուների տեսքը (գույները, տառատեսակը և այլն) ինչպես է փոխվում՝ տրված պայմանի կատարման դեպքում։
