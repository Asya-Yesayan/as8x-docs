# ViewExtension.DynamicHandler հատկություն

## Նկարագիր

**Դաս՝** [ViewExtension](../ViewExtension.md)

```c#
public IDynamicHandler DynamicHandler { get; }
```

Վերադարձնում է ընդլայնման դինամիկ մշակիչը։ Դինամիկ մշակիչը սահմանում է տողերի վրա կատարվող գործողությունները՝ [BeforeExecute](../Methods/BeforeExecute.md), [ProcessRow](../Methods/ProcessRow.md) և [AfterExecute](../Methods/AfterExecute.md) մեթոդների միջոցով։
