# Սերվիսային ասինխրոն պրոցեսներ դիտելու ձև

## Ներածություն

8X սերվիսում կատարվող պրոցեսների դիտարկման և կառավարման համար ավելացվել են՝ **«8X Web ծառայություններ»** և **«Սերվիսային ասինխրոն պրոցեսներ»** դիտելու ձևերը։ Նշված դիտելու ձևերը հասանելի են **«Ադմինիստրատորի աշտ» -> «Համակարգային աշխատանքներ»** թղթապանակից։

![processes_view](images/processes_view.png)

{: .note }
**«Սերվիսային ասինխրոն պրոցեսներ»** դիտելու ձևը տալիս է հնարավորություն դիտել և ընդհատել սերվիսների պրոցեսները (տվյալների աղբյուրի, տվյալների մշակման հարցման (DPR) առաջադրանքների կատարում, փաստաթղթի հեռացում և գրանցում, ․․․)։

Վերոնշյալ դիտելու ձևի նախնական ֆիլտրման պատուհանում հնարավոր է ֆիլտրել այն ըստ վեբ ծառայության (սերվիսի) id-ի։ Չֆիլտրելու դեպքում կատարման արդյունքում կցուցադրվեն բոլոր սերվիսների ասինխրոն պրոցեսները։

![processes_dialog](images/processes_dialog.png)

{: .note }
Դիտելու ձևում նաև ցուցադրվում են տողեր, որտեղ բացակայում է պրոցեսի ինֆորմացիան (id, անվանում, տիպ ...): Դրանք ներկայացնում են սերվիսներում առկա ընթացիկ սեսսիաները։

## Կոնտեքստային ֆունկցիաներ

### Ընդհատել պրոցես(ներ)ը

«Ընդհատել պրոցեսը», «Ընդհատել պրոցեսները» կոնտեքստային ֆունկցիաները տալիս են հնարավորություն ընդհատել ընտրված սերվիսային ասինխրոն պրոցես(ներ)ը (տվյալների աղբյուրի, տվյալների մշակման հարցման (DPR), առաջադրանքի կատարում, փաստաթղթի հեռացում և գրանցում, ․․․)։ 

Ֆունկցիաների կանչի արդյունքում կընդհատվեն այն պրոցեսները, որոնց համար նշված է պրոցեսի Id-ն (պրոցես Id չունեցող տողերը հանդիսանում են ընթացիկ սեսսիաներ)։

![stop_process](images/stop_processes.png)

Պրոցեսը գործարկող օգտագործողը ֆունկցիայի կանչի արդյունքում ստանում է սխալի հաղորդագրություն, որտեղ նշվում է X օգտագործողի կողմից ընդհատման փաստը։

![stop_process_error_message](images/stop_process_error_message.png)

### Ընդհատել պրոցես(ներ)ը և հեռացնել արդյունք(ներ)ը

«Ընդհատել պրոցեսը և հեռացնել արդյունքը», «Ընդհատել պրոցեսները և հեռացնել արդյունքները» կոնտեքստային ֆունկցիաները տալիս են հնարավորություն ընդհատել սերվիսային ասինխրոն պրոցես(ներ)ը (տվյալների աղբյուրի, տվյալների մշակման հարցման (DPR), առաջադրանքի կատարում, փաստաթղթի հեռացում և գրանցում, ․․․) և հեռացնել պրոցես(ներ)ի քեշը սերվիսից։

Կոնտեքստային ֆունկցիաները կիրառելի են «Սերվիսային ասինխրոն պրոցեսներ» դիտելու ձևի այն տողերի համար, որոնց համար նշված է պրոցեսի Id-ն (պրոցես Id չունեցող տողերը հանդիսանում են ընթացիկ սեսսիաներ)։

![stop_processes_and_remove_results](images/stop_processes_and_remove_results.png)

Պրոցեսը գործարկող օգտագործողը ֆունկցիայի կանչի արդյունքում ստանում է սխալի հաղորդագրություն, որտեղ նշվում է X օգտագործողի կողմից ընդհատման փաստը։

![stop_process_error_message](images/stop_process_error_message.png)

### Ընդհատել սեսիան

Այս կոնտեքստային ֆունկցիան տալիս է հնարավորություն ընդհատել օգտագործողի սեսիան։ Սեսիայի ընդհատման արդյունքում հեռացվում են սեսսիայի ընթացքում ստեղծված ժամանակավոր ֆայլերը, ընդհատվում ընթացիկ բոլոր պրոցեսսները (տվյալների աղբյուրի, տվյալների մշակման հարցման (DPR), առաջադրանքի կատարում, փաստաթղթի հեռացում և գրանցում, ․․․)։

![stop_session](images/stop_session.png)

### Դիտել մանրամասն նկարագրությունը

Վերադարձնում է «Սերվիսային ասինխրոն պրոցեսներ» դիտելու ձևի տողին համապատասխան պրոցեսի նկարագրությունը (վիճակ, վիճակի անվանում, կատարման սկզբի ամսաթիվ/ժամանակ, ․․․)։ Նկարագրությունը պահվում է json ֆորմատի ֆայլում և ցուցադրվում։

![view_process_details](images/view_process_details.png)
