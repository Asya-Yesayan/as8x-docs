# Սերվիսային ասինխրոն պրոցեսներ դիտելու ձև

## Ներածություն

Ծրագիր մուտք գործելիս սերվիսում բացվում է սեսիա, որը պարունակում է մուտք գործած օգտագործողի տվյալները և ծրագրի աշխատանքի որոշ կարգավորումներ։ Սեսիան ավտոմատ կերպով փակվում է ծրագրից դուրս գալիս։

Պրոցեսը սերվիսում կատարվող ասինխրոն գործողություն է (օրինակ՝ տվյալների աղբյուրի, DPR-ի, առաջադրանքի կատարում, փաստաթղթի հեռացում և գրանցում, ․․․), որը գործարկվում է տվյալ սեսիայի շրջանակում։ Մեկ սեսիայի ընթացքում կարող են միաժամանակ գործարկվել մի քանի պրոցեսներ։

8X սերվիսում կատարվող պրոցեսների դիտարկման և կառավարման համար ավելացվել են՝ **«8X Web ծառայություններ»** և **«Սերվիսային ասինխրոն պրոցեսներ»** դիտելու ձևերը։ Նշված դիտելու ձևերը հասանելի են **«Ադմինիստրատորի աշտ» -> «Համակարգային աշխատանքներ»** թղթապանակից։

![processes_view](images/processes_view.png)

{: .note }
**«Սերվիսային ասինխրոն պրոցեսներ»** դիտելու ձևը տալիս է հնարավորություն դիտել և ընդհատել սերվիսների պրոցեսները (տվյալների աղբյուրի, DPR-ի, առաջադրանքի կատարում, փաստաթղթի հեռացում և գրանցում, ․․․)։

Վերոնշյալ դիտելու ձևի նախնական ֆիլտրման պատուհանում հնարավոր է ֆիլտրել այն ըստ վեբ ծառայության (սերվիսի) id-ի։ Չֆիլտրելու դեպքում կատարման արդյունքում կցուցադրվեն բոլոր սերվիսների ասինխրոն պրոցեսները։

![processes_dialog](images/processes_dialog.png)

{: .note }
Դիտելու ձևում նաև ցուցադրվում են տողեր, որտեղ բացակայում է պրոցեսի ինֆորմացիան («Պրոցեսի նկարագրություն», «Պրոցեսի ID», ․․․ սյուների արժեքները դատարկ են): Այդ տողերը ներկայացնում են սերվիսներում առկա ընթացիկ սեսիաները։

## Կոնտեքստային ֆունկցիաներ

### Ընդհատել պրոցես(ներ)ը

«Ընդհատել պրոցես(ներ)ը» կոնտեքստային ֆունկցիաները տալիս են հնարավորություն ընդհատել ընտրված սերվիսային ասինխրոն պրոցես(ներ)ը  (տվյալների աղբյուրի, DPR-ի, առաջադրանքի կատարում, փաստաթղթի հեռացում և գրանցում, ․․․)։

Ֆունկցիաները հասանելի են դիտելու ձևի այն տողերի համար, որոնք ներկայացնում են պրոցես («Պրոցեսի ID» սյան արժեքը լրացված է՝ ի տարբերություն սեսիաների, որտեղ այն դատարկ է), և գտնվում է կատարման ընթացքում («Պրոցեսի վիճակ» սյան արժեքը Started է)։

![stop_process](images/stop_processes.png)

Պրոցեսը գործարկող օգտագործողը ֆունկցիայի կանչի արդյունքում ստանում է սխալի հաղորդագրություն, որտեղ նշվում է X օգտագործողի կողմից ընդհատման փաստը։

![stop_process_error_message](images/stop_process_error_message.png)

### Ընդհատել պրոցես(ներ)ը և հեռացնել արդյունք(ներ)ը

«Ընդհատել պրոցես(ներ)ը և հեռացնել արդյունք(ներ)ը» կոնտեքստային ֆունկցիաները տալիս են հնարավորություն ընդհատել սերվիսային ասինխրոն պրոցես(ներ)ը  (տվյալների աղբյուրի, DPR-ի, առաջադրանքի կատարում, փաստաթղթի հեռացում և գրանցում, ․․․) և հեռացնել պրոցես(ներ)ի քեշը սերվիսից։

Ֆունկցիաները հասանելի են դիտելու ձևի այն տողերի համար, որոնք ներկայացնում են պրոցես («Պրոցեսի ID» սյան արժեքը լրացված է՝ ի տարբերություն սեսիաների, որտեղ այն դատարկ է), և գտնվում է կատարման ընթացքում («Պրոցեսի վիճակ» սյան արժեքը Started է)։

![stop_processes_and_remove_results](images/stop_processes_and_remove_results.png)

Պրոցեսը գործարկող օգտագործողը ֆունկցիայի կանչի արդյունքում ստանում է սխալի հաղորդագրություն, որտեղ նշվում է X օգտագործողի կողմից ընդհատման փաստը։

![stop_process_error_message](images/stop_process_error_message.png)

### Ընդհատել սեսիան

Այս կոնտեքստային ֆունկցիան տալիս է հնարավորություն ընդհատել օգտագործողի սեսիան։ Սեսիայի ընդհատման արդյունքում հեռացվում են սեսիայի ընթացքում ստեղծված ժամանակավոր ֆայլերը, ընդհատվում ընթացիկ բոլոր պրոցեսսները  (տվյալների աղբյուրի, DPR-ի, առաջադրանքի կատարում, փաստաթղթի հեռացում և գրանցում, ․․․)։

Ֆունկցիան հասանելի է դիտելու ձևի այն տողերի համար, որոնք ներկայացնում են սեսիա («Պրոցեսի ID» սյան արժեքը դատարկ է՝ ի տարբերություն պրոցեսների, որտեղ այն լրացված է)։

![stop_session](images/stop_session.png)

### Դիտել մանրամասն նկարագրությունը

Վերադարձնում է «Սերվիսային ասինխրոն պրոցեսներ» դիտելու ձևի տողին համապատասխան պրոցեսի նկարագրությունը (վիճակ, վիճակի անվանում, կատարման սկզբի ամսաթիվ/ժամանակ, ․․․)։ Նկարագրությունը պահվում է json ֆորմատի ֆայլում և ցուցադրվում։

Ֆունկցիան հասանելի է դիտելու ձևի այն տողերի համար, որոնք ներկայացնում են պրոցես («Պրոցեսի ID» սյան արժեքը լրացված է՝ ի տարբերություն սեսիաների, որտեղ այն դատարկ է)։

![view_process_details](images/view_process_details.png)

```json
{
    "State": 4,
    "StateName": "Started",
    "StartDate": "2026-02-11T12:35:54.4411466+04:00",
    "Details": []
}
```
