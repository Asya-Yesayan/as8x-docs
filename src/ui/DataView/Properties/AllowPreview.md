# DataView.AllowPreview հատկություն

## Նկարագիր

**Դաս՝** [DataView](../DataView.md)

```c#
public virtual bool AllowPreview { get; }
```

Սահմանում է դիտելու ձևի ընթացիկ տողում պարունակվող փաստաթղթի տպելու ձևանմուշի նախնական դիտման իրավասությունը: Հատկության լռությամբ արժեքը համընկնում է IsDocumentBased հատկության արժեքի հետ։

Ֆունկցիոնալությունը ընթացիկ պահին իրականացված չէ։
