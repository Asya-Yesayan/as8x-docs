# DataView.DialogSupported հատկություն

## Նկարագիր

**Դաս՝** [DataView](../DataView.md)

```c#
public virtual bool DialogSupported { get; }
```

Սահմանում է՝ արդյոք դիտելու ձևը սատարում է նախնական ֆիլտրման դիալոգի ցուցադրումը։ Հատկության լռությամբ արժեքը false է։

Հատկության true արժեքի դեպքում մինչև տվյալների բեռնումը ցուցադրվում է նախնական ֆիլտրման դիալոգը (սահմանվում է [CreateDialog](../Methods/CreateDialog.md) մեթոդի միջոցով)։
