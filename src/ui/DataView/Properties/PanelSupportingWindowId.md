# DataView.PanelSupportingWindowId հատկություն

## Նկարագիր

**Դաս՝** [DataView](../DataView.md)

```c#
public int? PanelSupportingWindowId { get; private set; }
```

Վերադարձնում է դիտելու ձևի վահանակը պարունակող պատուհանի ներքին նույնականացման համարը (id)։ Արժեքավորվում է դիտելու ձևը ցուցադրելիս՝ [ShowBrowser](../Methods/ShowBrowser.md) մեթոդի կանչի ժամանակ։
