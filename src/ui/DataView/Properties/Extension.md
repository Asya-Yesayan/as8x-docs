# DataView.Extension հատկություն

## Նկարագիր

**Դաս՝** [DataView](../DataView.md)

```c#
public IDataViewExtension Extension { get; set; }
```

Վերադարձնում կամ նշանակում է դիտելու ձևին կցված ընդլայնումը։ Ընդլայնման միջոցով հնարավոր է փոփոխել դիտելու ձևի սյուները, կցել տվյալների աղբյուրի ընդլայնիչ և կիրառել այլ կարգավորումներ։ Ընդլայնումը կիրառվում է [ApplyExtension](../Methods/ApplyExtension.md) մեթոդի միջոցով։
