# DataView.MayContainDeletedDocuments հատկություն

## Նկարագիր

**Դաս՝** [DataView](../DataView.md)

```c#
public virtual bool MayContainDeletedDocuments { get; }
```

Սահմանում է՝ արդյոք դիտելու ձևը կարող է պարունակել հեռացված փաստաթղթեր։ Հատկության լռությամբ արժեքը false է։

Հատկության true արժեքի դեպքում դիտելու ձևի տողից փաստաթուղթ բացելիս թույլատրվում է բացել նաև հեռացված փաստաթղթերը։
