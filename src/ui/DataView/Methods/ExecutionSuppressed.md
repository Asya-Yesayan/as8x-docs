# DataView.ExecutionSuppressed() մեթոդ

## Նկարագիր

**Դաս՝** [DataView](../DataView.md)

```c#
public virtual void ExecutionSuppressed()
```

Նախատեսված է դիտելու ձևի կատարման դադարեցման դեպքում կատարվող գործողությունների մշակման համար։

Կանչվում է SuppressExecution հատկության true արժեքի դեպքում, որի արդյունքում դիտելու ձևի տվյալների բեռնումը դադարեցվում է։ 

Մեթոդը չմշակելու դեպքում ոչ մի գործողություն չի կատարվում։
