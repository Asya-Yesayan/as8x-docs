# DataView.ExecutionSuppressed() մեթոդ

## Նկարագիր

**Դաս՝** [DataView](../DataView.md)

```c#
public virtual void ExecutionSuppressed()
```

Նախատեսված է դիտելու ձևի կատարման ճնշման դեպքում կատարվող գործողությունների համար։

Կանչվում է SuppressExecution հատկության true արժեքի դեպքում, երբ դիտելու ձևի տվյալների բեռնումը ճնշվում է։ Մեթոդը չմշակելու դեպքում ոչ մի գործողություն չի կատարվում։
