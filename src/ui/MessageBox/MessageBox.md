# MessageBox դաս նկարագրություն

## Ներածություն

MessageBox դասը նախատեսված է հաղորդագրության մոդալ պատուհան ցուցադրելու և օգտագործողի ընտրած կոճակին համապատասխան արդյունքը ստանալու համար։

## Մեթոդներ

| Անվանում | Նկարագրություն |
|----------|----------------|
| [Show(string, string, MessageBoxButton, MessageBoxImage, MessageBoxResult, Window, int, string, bool)](Methods/Show.md) | Ցուցադրում է հաղորդագրության մոդալ պատուհանը և վերադարձնում է օգտագործողի ընտրած կոճակին համապատասխան արդյունքը (**MessageBoxResult**)։ |
| [ShowAndGetResult(string, string, MessageBoxButton, MessageBoxImage, MessageBoxResult, Window, int, string, bool)](Methods/ShowAndGetResult.md) | Ցուցադրում է հաղորդագրության մոդալ պատուհանը և վերադարձնում է օգտագործողի ընտրած կոճակին համապատասխան արդյունքը(**MessageBoxInfoResult**):|
