# IGrid.CopyData(IGrid) մեթոդ

```c#
public void CopyData(IGrid grid);
```

Պատճենում է grid պարամետրում տրված աղյուսակի պարունակությունը ընթացիկ աղյուսակ։ Պատճենվող և ընթացիկ աղյուսակների տիպերի անհամապատասխանության դեպքում առաջանում է սխալ։

**Պարամետրեր**

| Անվանում        | Տվյալների տիպ        | Լռությամբ արժեք | Նկարագրություն |
|-----------------|----------------------|----------------|----------------|
|  grid           |  IGrid                | -             | Պատճենվող աղյուսակի նկարագրությունը։ |
