---
title: DataSource.Client հատկություն
---

```c#
public ApiClient Client { get; set; }
```

Վերադարձնում կամ նշանակում է [ApiClient](../../../Types/ApiClient/ApiClient.md) դասի օբյեկտ, որը նախատեսված է տվյալների աղբյուրի կատարման ընթացքում կլիենտից դեպի սերվեր անհրաժեշտ հարցումներ կատարելու համար։

Այս հատկությունը հարկավոր է լրացնել DataSource դասի օբյեկտը ստեղծելուց անմիջապես հետո։
